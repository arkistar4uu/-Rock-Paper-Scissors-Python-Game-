# 🪨 Rock! 📄 Paper!! ✂️ Scissors!!! – Python Game  

Welcome to **Rock-Paper-Scissors**, a fun and interactive game built with Python! Challenge the computer and test your luck in this classic game.  

---

## 🎮 How to Play?  
1. Run the script in a Python environment.  
2. Enter your move:  
   - **r** for Rock  
   - **p** for Paper  
   - **s** for Scissors  
3. The computer will randomly pick a move.  
4. The winner is determined based on the following rules:  
   - 🏆 **Rock crushes Scissors**  
   - ✂️ **Scissors cuts Paper**  
   - 📄 **Paper covers Rock**  
5. The game tracks your score against the computer!  
6. You can choose to **continue** or **exit** after each round.  

---

## 🛠 Features  
✅ Simple **command-line** interface  
✅ **Randomized AI opponent** 🤖  
✅ **Win/Loss messages** with fun emojis 🎉  
✅ **Score Tracking** 🏅  
✅ **Replay Option** after each round
